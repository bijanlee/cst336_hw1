Android Application Development

Android application development, like iOS development, is a subset of smartphone app development. In this overview, we will discuss native Android App development with Android Studio and Java, not cross-platform solutions such as Cordova or Xamarin. Android development is done inside of Android Studio which was developeed by Google on top of the Intellij IDE and designed specifically for the development of Android Applications. Prior to the release of Android Studio, Android app development was done inside of Eclipse. Android applications are written in Java and utilize XML for its layouts.
