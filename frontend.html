Frontend Development

Frontend web development is also referred to as Client side development, since the frontend technologies all run in the browsers, or client. The main technologies that front end developers will be using are HTML, CSS, and Javascript. Modern frontend development often leverages the use of frameworks. Currently, three of the most popular frontend frameworks are Angular, Vue, and React.
